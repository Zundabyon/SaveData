## 本サービスの概要（700 文字以内）

SaveData は、ゲームの思い出を「年齢 × 時間軸」で記録・可視化できる
ノスタルジー型サービスです。

多くのゲーム管理サービスが「何をどれだけ遊んだか」に焦点を当てる一方で、
本サービスは「いつ・何歳の頃に・どんな気持ちで遊んだか」という人生に紐づく
体験の記録に価値を置いています。

ユーザーは生年を登録し、遊んだゲームとその時の年齢、印象に残った感情や
一言メモを入力することで、自分だけのゲーム年表を作成できます。
年表を見ることで、ゲーム体験を通して自分の人生を振り返ることができ、
「この頃こんなゲームをしていた」という記憶を再体験できます。

主な想定ユーザーは、30 代前後の社会人で、
子ども時代から現在まで幅広いゲーム体験を持ち、
懐かしさや思い出を大切にしたいと感じている層です。
限られたプレイ時間の中でも、
過去のゲーム体験を価値ある記憶として残したいニーズに応えます。

また、同じゲームを遊んだ他ユーザーの年齢分布を閲覧することで、
「みんなもこの時期に遊んでいた」という共感や、
世代ごとの受け取り方の違いを発見できます。

SaveData は、ゲームを単なる娯楽ではなく「人生の記憶」として保存・共有できる
新しい体験を提供するサービスです。

### MVP リリース時の機能

- ユーザー登録 / ログイン
- 生年登録
- ゲーム登録機能（タイトル・プレイ時期・一言メモ）
- 年齢 × ゲーム年表表示

この時点では他ユーザーとの共有機能はありません。
本リリース時に実装予定です。

### テーブルの詳細

#### USERS（ユーザー個人の基本情報

| カラム名           | 型       | 説明                   | 備考                    |
| ------------------ | -------- | ---------------------- | ----------------------- |
| id                 | int PK   | ユーザー ID            | 主キー                  |
| email              | string   | メールアドレス         | ログイン認証用 / UNIQUE |
| name               | string   | 表示名                 |                         |
| birthday           | string   | 生年月日               | 例：1985-01-23          |
| gender             | string   | 性別                   | 任意入力                |
| encrypted_password | string   | 暗号化されたパスワード | 認証                    |
| created_id         | datetime | 作成日時               |                         |
| updated_id         | datetime | 更新日時               |                         |

#### STATUSES（ユーザー個人の詳細情報）

| カラム名       | 型       | 説明                 | 備考                   |
| -------------- | -------- | -------------------- | ---------------------- |
| id             | int PK   | ID                   | 主キー                 |
| user_id        | int FK   | ユーザー ID          | users.id へ外部キー    |
| like_gamehard  | string   | 好きなゲームハード   | 例：PS2 / Switch       |
| like_gametype  | string   | 好きなゲームジャンル | 例：RPG / ACT          |
| own_title      | string   | 称号                 | 自由記述               |
| self_introduce | text     | 自己紹介             | 自由文                 |
| salt           | string   | ソルト（認証補助）   | 認証実装次第で不要可能 |
| role           | int      | ユーザー権限         | 例：admin=1 / user=0   |
| created_id     | datetime | 作成日時             |                        |
| updated_id     | datetime | 更新日時             |                        |

#### GAMES（登録しているゲーム）

| カラム名   | 型       | 説明               | 備考          |
| ---------- | -------- | ------------------ | ------------- |
| id         | int PK   | ID                 | 主キー        |
| title      | string   | 登録ゲームタイトル |               |
| hardware   | string   | 登録ハード         | 例：SFC / PS2 |
| type       | string   | ジャンル           | 例：RPG / ACT |
| recommend  | string   | おすすめ度         | ５段階評価    |
| difficulty | string   | 難易度             | ５段階評価    |
| memo       | text     | ゲームの思い出     | 自由記述      |
| created_id | datetime | 作成日時           |               |
| updated_id | datetime | 更新日時           |               |

#### BOARDS（年表単位のコンテナ）

| カラム名   | 型       | 説明        | 備考             |
| ---------- | -------- | ----------- | ---------------- |
| id         | int PK   | ID          | 主キー           |
| user_id    | int FK   | ユーザー ID | users.id への FK |
| game_id    | int FK   | ゲーム ID   | users.id への FK |
| event_id   | int FK   | イベント ID | event.id への FK |
| created_id | datetime | 作成日時    |                  |
| updated_id | datetime | 更新日時    |                  |

#### BOARDS_GAMES（年表とゲームを結ぶ中間テーブル）

| カラム名 | 型     | 説明      | 備考      |
| -------- | ------ | --------- | --------- |
| id       | int PK | ID        | 主キー    |
| board_id | int FK | 年表 ID   | boards.id |
| game_id  | int FK | ゲーム ID | games.id  |

#### EVENTS (イベントの登録)

| カラム名   | 型       | 説明           | 備考                |
| ---------- | -------- | -------------- | ------------------- |
| id         | int PK   | ID             | 主キー              |
| title      | string   | イベント内容   | 例：1989 年 GB 発売 |
| event_year | datetime | イベント年     |                     |
| event_type | string   | イベントタイプ |                     |

#### BOARDS_EVENTS（年表とイベントをつなぐ中間テーブル）

| カラム名 | 型     | 説明        | 備考      |
| -------- | ------ | ----------- | --------- |
| id       | int PK | ID          | 主キー    |
| board_id | int FK | 年表 ID     | boards.id |
| event_id | int FK | イベント ID | events.id |
